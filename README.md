# oozo_box

Clone this repo to install OOZO.tv Social Media Television to your Raspberry Pi

1) Git clone https://github.com/OOZOtv/OOZObox
2) sudo chmod +x /home/pi/OOZObox/oozo.sh expand.sh
3) cd /home/pi/OOZObox
4) ./expand.sh #This will expand your filesystem and reboot your Raspberry
5) cd /home/pi/OOZObox
6) ./oozo.sh #This will take awhile to finish

Done
